# Trader Sentiment Analysis

## Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Objective
The objective is to explore how Bitcoin market sentiment affects trader profitability and identify insights that can support smarter trading strategies.

## Dataset
- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Date Conversion
4. Dataset Merging
5. Exploratory Data Analysis (EDA)
6. Data Visualization
7. Performance Analysis
8. Business Insights

## Project Files
- `Trader_Sentiment_Analysis.ipynb`
- `Trader Performance Analysis Using Bitcoin Market Sentiment1.pdf`
- `requirements.txt`

## Key Insights
- Compared trader performance under Fear and Greed market conditions.
- Analyzed trading behavior, profitability, and trade distribution.
- Identified trends that can support better trading decisions.
